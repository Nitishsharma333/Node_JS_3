This API demonstrates seat locking and booking to prevent double reservations.

Available Endpoints:

  • GET /api/seats - View all seats
  • POST /api/seats/lock/:id - Lock a seat for 1 minute
  • POST /api/seats/book/:id - Confirm booking (must be locked first)

Run the server using node src/app.js and use Postman or cURL to test the API.
